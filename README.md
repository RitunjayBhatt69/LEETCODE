Example Problem: Regular Expression Matching

Problem Statement

Implement regular expression matching with support for . and * where:

. Matches any single character.

* Matches zero or more of the preceding element.

Example Solutions

Input:

s = "aa"
p = "a"

Output:

False

Input:

s = "aa"
p = "a*"

Output:

True
